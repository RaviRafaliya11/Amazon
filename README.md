# Amazon Clone

An e-commerce experience inspired by Amazon with real-time backend and Stripe Checkout.

## 🚀 Features
- Product listings with images and price
- Add to cart & checkout
- Stripe Checkout & Webhooks
- Firebase backend
- Auth with NextAuth

## 🛠️ Tech Stack
- React
- Next.js
- Firestore
- Stripe
- NextAuth

## 📦 Setup
```bash
npm install
npm run dev
